# UFCLite — аркадный файтинг (UE5.7.4, C++)

## Концепция
Простой 3D-файтинг в духе UFC. Два бойца, удары руками/ногами, блок, нокаут, кровь.

## Управление (UFC-style chorded)
### Клавиатура
| Действие | Кнопка |
|---|---|
| Джеб | X |
| Кросс | Y |
| Хук | B |
| Апперкот | A |
| Лоу-кик | X (R2 held on gamepad) |
| Мидл-кик | Y (R2 held) |
| Хай-кик | B (R2 held) |
| Кик в корпус | A (R2 held) |
| Блок | Left Shift |
| Передвижение | WASD (world-space) |

### Геймпад (когда заработает)
| Действие | Кнопка |
|---|---|
| Ведущие удары | □ / △ / ○ / ✕ |
| Ноги | R2 + □ / △ / ○ / ✕ |
| Блок | R1 |
| Передвижение | Левый стик |
