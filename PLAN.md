# UFCLite — аркадный файтинг (UE5.7.4, C++)

## Концепция
Простой 3D-файтинг в духе UFC. Выбор бойца, бой до нокаута, кровь и увечья.
Без меню/прогрессии — только зашёл и подрался.

## Механики
- 2 игрока (локальный мультиплеер, геймпад + клавиатура)
- Удары руками (джеб, кросс, хук, апперкот)
- Удары ногами (лоу-кик, мидл-кик, хай-кик)
- Блок (удержание кнопки)
- Здоровье (2 полоски: здоровье и выносливость)
- Нокаут при обнулении здоровья
- Кровь на модели бойца (decals + particle system)
- Увечья (визуальные: синяки, кровоподтёки на скин)

## Управление (геймпад Xbox)
| Действие | Кнопка |
|---|---|
| Джеб | X |
| Кросс | Y |
| Хук | B |
| Апперкот | A |
| Лоу-кик | LB |
| Мидл-кик | RB |
| Хай-кик | LB+RB |
| Блок | LT |
| Передвижение | Левый стик |

## Файлы проекта

### C++ классы
- `MainPlayerController` — управление вводом
- `FighterCharacter` — персонаж, здоровье, анимации
- `FighterAnimInstance` — анимационный блупринт
- `HealthComponent` — здоровье и выносливость
- `DamageSystem` — расчёт урона
- `FightGameMode` — логика боя (раунды, нокаут)
- `FightHUD` — интерфейс (полоски здоровья)
- `BloodSystem` — декали крови на моделях
- `FighterSelectWidget` — выбор персонажа

### Ассеты (нужно скачать)

## Порядок реализации
1. База: GameMode, персонаж, передвижение
2. Система ударов: анимации, хитбоксы, урон
3. Здоровье и нокаут
4. Кровь и увечья (VFX)
5. Экран выбора бойца
6. Геймпад
